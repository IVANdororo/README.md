# README.md
Программа предназначена для автоматического расчёта стоимости ремонта мобильных телефонов в сервисном центре. Позволяет определить итоговую цену с учётом:
- модели устройств
- типа неисправности
- стоимсоть запчастей
- срочность выполнения
- дополнительных услуг
- скидок по промокодам
